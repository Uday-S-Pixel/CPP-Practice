# C++ Programming – Exercise Questions (File Handling & Exception Handling)

The questions for this exercise are listed below.

## Questions

1. Write a program to implement **I/O operations on characters**.
   The operations include:

   * Inputting a string
   * Calculating length of the string
   * Storing the string in a file
   * Fetching the stored characters from the file

2. Write a program to **copy the contents of one file to another**.

3. Write a program to perform **binary I/O operations on a file**
   (i.e., write and read an object of a structure/class to/from a file).

4. Write a program for **reading and writing data using command line arguments**.

5. Write a program to implement **exception handling with multiple catch statements**.

6. Write a program to implement **exception handling with throw restrictions**.
