# C++ Programming – Exercise Questions (Templates)

The questions for this exercise are listed below.

## Questions

1. Write a **function template** that will sort an array of implicit types like:

   * `int`
   * `float`
   * `char`

   It should also be able to sort **user-defined objects** like:

   * strings
   * date

   *(Note: Necessary classes should include operator overloading.)*

2. Write a C++ program to implement a **function template** for finding the **minimum value**.
