# C++ Programming – Exercise Questions

The questions for this exercise are listed below.

## Questions

1. Write a program to display the message:  
   **"Hello, Welcome to C++ Programming."**

2. Write a program to print the **name and roll number** of a student:
   - in two separate lines.

3. Write a program to print a **name** by:
   - reading it from the user,
   - assigning it to a variable, and
   - initializing it with an appropriate prompt.

4. Write a program to print the following **personal details** by taking input from the user:
   - Name  
   - Surname (single character)  
   - Total Marks  
   - Gender (M/F)  
   - Result (P/F)

5. Write a program to convert **Centigrade (C)** into **Fahrenheit (F)**.  

   Formula:

   C = (F - 32) / 1.8

6. Write a program that declares **two integers** and determines whether the **first integer is a multiple of the second**.  
   *(Hint: Use the remainder operator `%`.)*

7. Write a program that prompts the user to enter two integers (`val1` and `val2`) and find:
   - Largest value  
   - Sum  
   - Difference  
   - Product  
   - Ratio

8. Write a program that prompts the user to enter **three integer values** and outputs them in **ascending order**, separated by commas.

   **Example:**

   Input:
   10 4 6

   Output:
   4,6,10

   Input:
   4 5 4

   Output:
   4,4,5
