# C++ Programming – Exercise Questions (OOP)

The questions for this exercise are listed below.

## Questions

1. Write a Program to design a class having static member function named `showcount()`
   which displays the number of objects created of the class.

2. Write a Program using class to process a **Shopping List** for a Departmental Store.
   The list should include:

   * Code No
   * Price

   Perform operations like:

   * Adding items
   * Deleting items
   * Printing total value of an order

3. Write a Program which creates and uses an **array of objects**
   (Example: list of Managers of a Company having details such as Name, Age, etc.)

4. Write a Program to find **Maximum out of Two Numbers** using a **friend function**.

   * One number is a member of one class
   * The other number is a member of another class

5. Write a Program to **swap private data members** of two classes (`class_1`, `class_2`)
   using a **friend function**.

6. Write a Program to design a class `complex` to represent complex numbers.

   * Use an external function (friend function) to add two complex numbers
   * Return an object representing the sum

7. Write a Program using a **copy constructor** to copy data of one object to another.

8. Write a Program to **allocate memory dynamically** for an object using a class constructor.

9. Write a Program to **overload operators** like `*`, `<<`, `>>` using friend functions
   for a class `vector`.
